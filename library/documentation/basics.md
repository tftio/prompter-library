# Documentation Basics

1. All documentation *MUST* be in Github Flavored Markdown format, with a `.md` file extension.
2. All markdown documentation *MUST* pass `markdownlint-cli2` linting; you can execute the command with `npx markdownlint-cli2 --fix $filename`;
3. All documentation *MUST* have an up-to-date entry in the `docs/map.md` document map file.
